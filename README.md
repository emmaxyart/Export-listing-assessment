
# Expert Listing Assessment

This is a Next.js, Tailwind CSS project, Radix UI project.

# Overview

The Expert Listing Project is a web application designed to showcase a directory of experts. It features a modern, user-friendly interface with various components that allow for easy navigation and interaction. The project is built with Next.js and utilizes a range of UI components to provide a rich user experience.

# Key Features

- Listings: View a comprehensive list of experts with detailed profiles.
- Interactive UI: A set of well-designed UI components for a seamless user experience.
- Responsive Design: The application is designed to work on various devices, including desktops and mobile phones.
- Theme: Supports theme switching for a personalized look and feel.

# The Overall Project Structure

The project is organized into the following directories:

- app: Contains the core application files, including pages and layouts.
- components: Houses all the reusable UI components used throughout the application.
- public: Stores static assets like images and icons.

# Components

The project includes a wide array of components  for the application. Below is a list of the main components used:

# Core Components
- `listings-overview.tsx`: Displays an overview of all the listings.
- `featured-listings.tsx`: Highlights most clicked, most watchlist and Hottest listings.
- `sales-overview.tsx`: Provides an overview of sales data.
- `users-overview.tsx`: Shows an overview of user statistics.
- `navigation.tsx`: The main navigation bar for the application.
- `mobile-menu.tsx`: A responsive menu for mobile devices.
- `profile-dropdown.tsx`: A dropdown menu for user profiles.
- `theme-provider.tsx`: Manages the application's theming.

# UI Kit

# Installation

1. Clone the repo
   git clone https://github.com/your-username/expert-listing-project.git

2. Install NPM packages
   npm install
  

# Running the Application

To run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

 Deployed on Vercel: https://expert-listing-assessment.vercel.app
